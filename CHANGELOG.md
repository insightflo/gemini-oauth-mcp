# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-01-28

### Added
- Initial release
- Google OAuth 2.0 authentication with PKCE
- Multi-account support with rotation
- Rate limit handling with automatic account switching
- MCP Tools:
  - auth_login: Add Google account
  - auth_list: List registered accounts
  - auth_remove: Remove account
  - auth_status: Show authentication status
  - chat: Chat with Gemini AI
  - generate_content: Generate content
  - quota_status: Show quota status
